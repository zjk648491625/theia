Class Path

On POSIX: ┌──────────────────────┬────────────┐ │ dir │ base │ ├──────┬ ├──────┬─────┤ │ root │ │ name │ ext │ " / home/user/dir / file .txt " └──────┴───────────────┴──────┴─────┘

On Windows: ┌──────────────────────┬────────────┐ │ dir │ base │ ├──────┬ ├──────┬─────┤ │ root │ │ name │ ext │ " /c: / home/user/dir / file .txt " └──────┴───────────────┴──────┴─────┘
